# Product Advocate

**Product Advocate** — минималистичное браузерное web-приложение для контроля сроков годности продуктов и товаров.

Проект решает практическую задачу учёта сроков хранения: пользователь добавляет товар, указывает дату покупки или производства и срок годности, после чего приложение автоматически рассчитывает дату истечения и показывает актуальный статус каждой позиции.

Все данные хранятся локально в браузере с использованием `localStorage`. Приложение не требует регистрации, серверной части и работает офлайн.

---

## Основные возможности

- Добавление товаров с датой начала и сроком годности
- Автоматический расчёт даты «годен до»
- Отображение оставшихся дней до истечения срока
- Статусы позиций:
  - «Ок»
  - «Скоро истекает»
  - «Просрочено»
- Фильтрация и сортировка списка по дате окончания

---

## Техническая концепция

- Чистый HTML, CSS и Vanilla JavaScript
- Отсутствие внешних зависимостей
- Хранение данных в `localStorage`
- Даты хранятся в формате `YYYY-MM-DD`, отображаются пользователю как `dd.mm.yyyy`
